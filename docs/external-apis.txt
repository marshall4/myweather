========================================
API 1: NOAA National Weather Service API
========================================

Base URL:     https://api.weather.gov
Auth:         None (public API)
Format:       GeoJSON (application/geo+json)
Cost:         Free

--- Endpoints Used ---

1. Points Lookup
   URL:     GET https://api.weather.gov/points/{lat},{lon}
   Purpose: Converts GPS coordinates into an NWS grid reference (gridId, gridX, gridY)
            and retrieves forecast zone, city, and state for display.
   Called:  lookupGridpoint(lat, lon) — also called by lookupByZip()
   Returns:
     - properties.gridId         → NWS grid office ID (e.g. "LWX")
     - properties.gridX          → Grid X coordinate
     - properties.gridY          → Grid Y coordinate
     - properties.forecastZone   → Zone ID (last path segment, e.g. "VAZ505")
     - properties.relativeLocation.properties.city
     - properties.relativeLocation.properties.state

2. Hourly Forecast
   URL:     GET https://api.weather.gov/gridpoints/{gridId}/{gridX},{gridY}/forecast/hourly
   Purpose: Fetches the current weather conditions from the first period of the hourly forecast.
   Called:  fetchCurrentWeather(location)
   Returns (first period):
     - temperature
     - temperatureUnit
     - windSpeed
     - windDirection
     - shortForecast
     - icon (URL)
     - relativeHumidity.value
     - probabilityOfPrecipitation.value

3. Extended Forecast
   URL:     GET https://api.weather.gov/gridpoints/{gridId}/{gridX},{gridY}/forecast
   Purpose: Fetches the multi-day forecast (all periods).
   Called:  fetchForecast(location)
   Returns (per period):
     - name
     - temperature
     - temperatureUnit
     - windSpeed
     - windDirection
     - shortForecast
     - isDaytime
     - icon (URL)

4. Active Weather Alerts
   URL:     GET https://api.weather.gov/alerts/active?zone={forecastZone}
   Purpose: Fetches any active weather alerts for the user's forecast zone.
   Called:  fetchAlerts(location)
   Returns (per alert in features[]):
     - properties.event
     - properties.headline
     - properties.severity
     - properties.onset
     - properties.expires
     - properties.description
     - properties.instruction

5. Weather Condition Icons
   URL:     https://api.weather.gov/icons/land/{timeOfDay}/{condition}?size=medium
   Purpose: Icon URLs are returned inline by the forecast/hourly endpoints (see "icon" field above).
            The Coil image library fetches these URLs directly to display condition icons in the UI.
   Called:  Coil AsyncImagePainter (not a direct app call)


========================================
API 2: Zippopotam.us
========================================

Base URL:     https://api.zippopotam.us
Auth:         None (public API)
Format:       JSON
Cost:         Free

--- Endpoints Used ---

1. ZIP Code Lookup
   URL:     GET https://api.zippopotam.us/us/{zip}
   Purpose: Converts a US ZIP code entered by the user into latitude/longitude coordinates,
            which are then passed to the NOAA points lookup to build the full LocationConfig.
   Called:  lookupByZip(zip)
   Returns:
     - places[0].latitude   → passed to lookupGridpoint()
     - places[0].longitude  → passed to lookupGridpoint()
